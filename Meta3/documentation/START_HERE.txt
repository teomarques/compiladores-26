KEY CONCEPTS IN 60 SECONDS
================================================================================

TYPE SYSTEM:
  Your compiler recognizes 6 types:
  • int, double, boolean (primitives)
  • String[] (array of strings)
  • void (return type)
  • undef (error/undefined - for error recovery)

SYMBOL TABLES:
  3 levels of symbols:
  • Class level: global variables and method names
  • Method level: formal parameters and local variables
  • All symbols are stored in declaration order

THREE PHASES:
  1. Symbol Table Building
     - Walk the AST
     - Extract all declarations
     - Build lookup tables

  2. Type Checking
     - Traverse expressions
     - Infer types recursively
     - Annotate AST nodes with types
     - Validate operations

  3. Error Reporting
     - Collect all errors
     - Report with line/column info
     - Continue analyzing despite errors

EXAMPLE OUTPUT:
  ===== Class Factorial Symbol Table =====
  factorial ( int ) int
  main ( String []) void

  ===== Method factorial ( int ) Symbol Table =====
  return  int
  n       int     param

  Program
  ..Identifier(Factorial)
  ..MethodDecl
  [... rest of annotated AST ...]
