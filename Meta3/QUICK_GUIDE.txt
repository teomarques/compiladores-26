SUPPORT & TROUBLESHOOTING
================================================================================

If something doesn't work:

1. Compilation errors?
   → Check INTEGRATION_GUIDE.md "Common Integration Issues" section

2. Symbol tables not printing?
   → Check build_class_symbol_table() implementation in symbol_table.c
   → Verify print_class_table() format matches specification

3. Type checking not working?
   → Start with literals first, then identifiers, then operations
   → Test each operator type separately

4. Output format wrong?
   → Compare with examples in INTEGRATION_GUIDE.md
   → Check printf format strings in print functions
